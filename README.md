# 📊 Basic information of the CLI computer

Projeto simples em Node.js para monitorar informações básicas do computador diretamente pelo terminal.

> ⚠️ Projeto de estudo

---

## 📌 Funcionalidades

- ✅ Exibe modelo do processador
- ✅ Uso da CPU em tempo real
- ✅ Memória RAM total
- ✅ Memória RAM livre
- ✅ Espaço total do disco
- ✅ Espaço livre do disco
- ✅ Espaço utilizado do disco
- ✅ Atualização automática a cada 1 segundo
- ✅ Interface colorida no terminal

---

## 📦 Instalação

Clone o projeto:

```bash
git clone https://github.com/RafaelHenriqu/Basic-information-of-the-CLI-computer.git
```

Entre na pasta:

```bash
cd Basic-information-of-the-CLI-computer
```

Instale as dependências:

```bash
npm install
```

---

## ▶️ Executando o projeto

```bash
node index.js
```

---

## 📚 Dependências

Instale manualmente caso necessário:

```bash
npm install os-utils diskspace
```
